## [공지사항 올리기 페이지]

- **제목:** 공지사항 제목 입력
- **내용:** 공지사항 내용 입력 (여러 줄 입력 가능)
- **대상:** 학교, 학년, 반 선택 (선택하지 않으면 전체 공지)
- [확인] [취소] 버튼

---

## [오늘의 숙제 올리기 페이지]

- **제목:** 숙제 제목 입력
- **마감일:** 숙제 마감일 선택
- **대상:** 학교, 학년, 반 선택
- **숙제 내용:**
  - [ ] 숙제 항목 텍스트 입력 ➕(버튼: 항목 추가)
  - [ ] 숙제 항목 텍스트 입력
- [확인] [취소] 버튼

---

## [개인 숙제 올리기 페이지]

- **제목:** 개인 숙제 제목 입력
- **마감일:** 숙제 마감일 선택
- **대상:** 학교, 학년, 반, 출석번호 선택
- **숙제 내용:**
  - [ ] 숙제 항목 텍스트 입력 ➕(버튼: 항목 추가)
  - [ ] 숙제 항목 텍스트 입력
- [확인] [취소] 버튼
